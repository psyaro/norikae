# CLAUDE.md

## プロジェクト概要

東京の地下鉄・JR・私鉄を対象にした障がい者割引運賃の経路探索アプリ。
index.html + version.js + sw.js で構成。v1.html に旧版を保存。

## アーキテクチャ

### UI
- SPA構成: #search / #results / #lines / #about のハッシュルーティング + ボトムナビ
- PWA: version.js でバージョン一元管理。sw.js が importScripts で参照し CACHE_NAME を自動連動
- 結果表示: 初期30件、「もっと見る」で全件。詳細は `<details>` 展開時に遅延レンダリング
- 経由地: 最大3経由地を動的追加。各レグ独立探索 → cross-product合成
- デフォルトソート: IC運賃（最安探索が主目的）

### グラフ構造
- 各駅がノード（869駅）、隣接駅間が辺（重み＝営業キロ）
- 同名駅は自動接続(0km)、異名乗換は `CROSS_TRANSFERS` で定義(0km)
- `lineFamily()`: JR在来線全25路線を同一系統 "JR" として統合（改札内乗換扱い）
- 都電荒川線(SA)と他路線の乗換辺は0.5km重み（路面電車のため徒歩距離あり）
- 駅コンプレックス: Union-Find で同一駅(grp)を管理。乗換判定・重複排除に使用

### 運賃計算
- 経路を会社(operator)ごとのセクションに分割し、各社の運賃テーブルで個別計算して合算
- JR特定区間: `JR_SPECIAL` に登録された駅ペアは距離ベース運賃より優先
- JR 100km超: 5割引（障害者割引）
- JR大回り乗車特例: `jrShortestDist()` でJRサブグラフの最短営業キロを求め、実経路より短ければ最短距離で運賃計算
- 地方交通線(八高線JP): 幹線運賃テーブルでは正しくないため運賃計算無効化（`LOCAL_JR_LINES`）
- メトロ↔JR乗継割引: 接続駅・対象駅が一致する場合 IC/きっぷとも10〜20円割引

### 特殊運賃区間
- **綾瀬〜北千住のみ**: きっぷ160円 / IC155円（JR運賃扱い、BFなし）
- **南北線 目黒〜白金高輪区間内のみ**: きっぷ180円 / IC178円（固定、BF込み）
- **メトロ通常運賃**: FARE_ZONESにBF10円込み。`getFare(km, noBF)` で制御

## 経路探索アルゴリズム（RAPTOR風 + Plateau + 最安運賃物量探索）

### 概要

4段構成のハイブリッド。1500msの時間予算内で多様な経路を生成し、最安運賃を発見する。

### アルゴリズム構成

1. **RAPTOR風ラウンド探索** (`roundShortest`): 状態(駅, 乗換回数k)のDijkstra。
   各到着ノードごとに乗換回数別の最短距離を求め、Pareto最適のみ採用
2. **Plateau法** (`plateauRoutes`): 出発側・到着側の最短経路木の共有区間(plateau)から
   構造的に異なる代替経路を生成
3. **運賃近似Dijkstra** (`fareApproxDijkstra`): 辺重み = 都営≈0・有料=実距離。
   各社を優先した固定バイアス4パターンで探索
4. **物量探索** (1500ms): 運賃近似+会社重みランダム化 と 距離+乗換ジッターDijkstra を交互に回し、全候補を運賃計算して最安を拾う

### 重複排除・フィルタ
- **routeShape**: lineFamily × 始終駅grpで署名。同一(shape, 距離, 乗換数)内は駅数最少を採用
- **素通り乗換除外** (`hasPassthroughTransfer`): 0距離乗換を2連続で跨ぎ異なる3系統をまたぐ経路を排除
- **パス正規化** (`cleanPath`): 先頭末尾の同一grp除去 + 直接辺のある同一grpチェーン中間ノード除去

### おすすめスコア

`score = distance + transfers × 3 + paidCross × 3 + toeiCross × 0.3`

| 項 | 内容 | 重み |
|---|---|---|
| `distance` | 営業キロ(km) | 1.0 |
| `transfers` | 乗換回数 | 1回 = 3km相当 |
| `paidCross` | 有料会社またぎ | 1回 = 3km相当 |
| `toeiCross` | 都営またぎ | 1回 = 0.3km相当 |

## 注意事項

- 大江戸線(E)は都庁前で環状部と放射部が分岐する3方向接続
- JR山手線(JY)は環状（有楽町→東京 0.8kmで閉じる）
- 鶴見線(JI)は浅野で海芝浦支線、武蔵白石で大川支線が分岐
- TX浅草は「浅草(TX)」として区別（メトロ/都営浅草とは別駅扱い）
- 相模線の入谷は「入谷(JR)」として日比谷線の入谷と区別
- 新宿(丸ノ内線⇔新宿線)、蔵前(浅草線⇔大江戸線)はFAR_NOTESで遠い乗換を警告
