# 東京障がい者割引乗り換え探索

営業キロに基づく最短経路探索アプリ。障がい者割引運賃で計算。

## デモ

https://psyaro.github.io/norikae/

## 機能

- SPA構成（検索/結果/路線一覧/情報の4ページ）+ ボトムナビ
- Dijkstra法＋辺/駅ブロック＋乗換コスト可変による複数経路探索
- 5種類のソート（おすすめ/運賃/距離/乗換回数/駅数）
- 路線・駅一覧ブラウズ（駅から出発/到着を直接設定）
- 経路詳細で経由駅を常時表示
- 会社をまたぐ場合は会社別に運賃を計算して合算
- 同名駅の自動接続 + 異名乗換駅の接続
- JR在来線全線を同一ファミリーとして統合（改札内乗換扱い）
- JR特定区間運賃の自動適用
- JR 100km超の障害者割引（5割引）
- 遠い乗換の警告表示
- 他社線乗り換え・大回り乗車の警告
- PWA対応（オフライン利用可）
