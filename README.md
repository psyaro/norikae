## 機能

- RAPTOR風ラウンド + Plateau法 + 運賃近似物量探索による多様な経路候補生成
- 経由地指定（最大3経由地）
- 5種類のソート（おすすめ/運賃/距離/乗換回数/駅数）、デフォルトIC運賃順
- JR大回り乗車特例（JR最短営業キロで自動計算）
- メトロ↔JR乗継割引（中野/西船橋/綾瀬/北千住接続で10〜20円）
- 綾瀬〜北千住特殊運賃（160円/IC155円）
- 南北線 目黒〜白金高輪区間の特殊運賃（180円/IC178円）
- 地方交通線（八高線）の運賃計算不可表示
- 会社をまたぐ場合は会社別に運賃を計算して合算
- 同名駅の自動接続 + 異名乗換駅の接続
- JR在来線全線を同一ファミリーとして統合（改札内乗換扱い）
- JR特定区間運賃の自動適用
- 素通り乗換の自動除外
- PWA対応（オフライン利用可）
